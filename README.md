# Crop Disease Detection System 🌱

## Project Description

The Crop Disease Detection System is a machine learning-based application designed to identify diseases in crop leaves. By analyzing images of plant leaves, the system detects possible diseases and helps farmers take preventive actions to protect their crops.

## Objectives

* Detect plant diseases from leaf images.
* Help farmers identify crop problems early.
* Improve crop productivity and reduce losses.

## Features

* Upload crop leaf images.
* Image processing and analysis.
* Disease prediction using machine learning.
* Displays detected disease and suggestions.

## Technologies Used

* Python
* Machine Learning
* OpenCV
* TensorFlow / Scikit-learn
* NumPy
* Pandas
* HTML / CSS (for interface)

## System Architecture

1. Image Input (Leaf Image)
2. Image Preprocessing
3. Feature Extraction
4. Disease Classification Model
5. Display Result

## Installation

1. Clone the repository
   git clone https://github.com/yourusername/crop-disease-detection.git

2. Navigate to the project folder
   cd crop-disease-detection

3. Install required libraries
   pip install -r requirements.txt

4. Run the program
   python app.py

## Dataset

The dataset consists of crop leaf images used to train and test the model for disease detection.

Example datasets:

* PlantVillage Dataset
* Custom collected crop images

## Output

The system predicts the disease present in the crop leaf and displays the result with possible recommendations.
